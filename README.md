# 🔐 Web Application Vulnerability Assessment – OWASP ZAP

## 📌 Introduction

This project presents a web application security assessment conducted as part of the Future Internship 2026 program. The analysis was performed using OWASP ZAP, an open-source penetration testing tool widely used to identify vulnerabilities in web applications.

The objective of this assessment is to analyze the security posture of a target web application by identifying common vulnerabilities such as SQL Injection, Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF), and missing security headers.

This report demonstrates how automated and manual testing techniques can be used to detect security flaws and provides insights into improving application security through proper implementation of security controls and best practices.
